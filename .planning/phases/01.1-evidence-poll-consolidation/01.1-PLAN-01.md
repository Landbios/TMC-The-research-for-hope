# Phase 01.1: Evidence Poll Consolidation (Plan 01)

## Goal
Consolidate the evidence polling logic currently housed in the legacy modal `GlobalPollOverlay.tsx` into the unified `NervalisOverlay` terminal to resolve UI conflicts and system notification failures.

## Context Constraints
- Rename the `POLLS` tab to `EVIDENCE`.
- This new tab will handle active evidence voting protocols AND act as a repository of all accepted evidence.
- Hybrid Alert Styling: Red theme when an evidence poll is active, clean blue when browsing accepted evidence.
- No forced tab switching. Tab icon just blinks triggering a global hud notification.
- Waiting State Feedback: suspenseful terminal readout showing consensus status.
- Privacy Polls move out of Nervalis to be a standalone pop-up.
- Assassin Polls stay strictly inside GlobalPollOverlay for roleplayers (Admin controlled logic).

## Requirements

- **REQ-EVIDENCE-01**: Rename `POLLS` tab to `EVIDENCE` and remove `LOG` tab in `NervalisOverlay.tsx`.
- **REQ-EVIDENCE-02**: Create `EvidenceTab.tsx` that switches between Active Poll (Red Theme + Waiting state feedback) and Historical Archive state.
- **REQ-EVIDENCE-03**: Remove Investigation Poll UI from `GlobalPollOverlay.tsx`.
- **REQ-EVIDENCE-04**: Decouple `PrivacyPollModal` and implement it outside `NervalisOverlay.tsx`.
