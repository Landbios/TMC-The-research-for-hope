# Phase 01.1: Evidence Poll Consolidation Summary

Consolidated investigation polling logic from the legacy `GlobalPollOverlay` into the unified `NervalisOverlay` terminal.

## Key Changes

- **Evidence Tab**: Created a new `EvidenceTab` that serves as both the active poll interface (with red-themed alerts) and the historical investigation log.
- **Terminal Integration**: Updated `NervalisOverlay` to include the `EVIDENCE` tab and removed the redundant `LOG` tab.
- **Logic Cleanup**: Removed investigation poll rendering from `GlobalPollOverlay` to eliminate UI conflicts.
- **Privacy Modal**: Decoupled `PrivacyPollModal` so it can be triggered as a standalone popup for room-blocking mechanics.

## Verification Results

- [x] **Tab Switch**: EVIDENCE tab appears in Nervalis and correctly switches between active/archive states.
- [x] **Conflict Resolution**: Investigation polls no longer appear in the global overlay.
- [x] **Privacy**: Privacy polls function independently of the terminal.

## Post-Completion Notes

This consolidation simplifies the investigate flow and ensures that all core investigation tools are centralized in Nervalis.
