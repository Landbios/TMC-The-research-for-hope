## Key Areas Investigation

### 1. Notification Synchronization
- **Approach**: Leverage the central Zustand store (`useTmaStore.ts`) to manage the `hasUnreadSignals` flag.
- **Trigger**: In `NervalisOverlay.tsx`, add a `useEffect` that monitors the `isOpen` state.
- **Pattern**:
  ```tsx
  useEffect(() => {
    if (isNervalisOpen) {
      setHasUnreadSignals(false);
    }
  }, [isNervalisOpen, setHasUnreadSignals]);
  ```
- **Reference**: Existing store structure in `src/store/useTmaStore.ts`.

### 2. Z-Index Management & Layering
- **Strategy**: Define a strict hierarchy in `tailwind.config.js` or via standard utility classes.
- **Hierarchy**:
  - `z-[100]`: NervalisOverlay (Draggable Terminal)
  - `z-[50]`: Fixed HUD elements (NervalisAccessButton, Room Title)
  - `z-[40]`: Standalone Modals (PrivacyPollModal, StealthEntryDialog)
  - `z-[30]`: Contextual UI (DiceAnimation, VNChatOverlay)
- **Constraint**: Use `framer-motion`'s `dragConstraints` to keep the terminal within the viewport.
- **Snapping**: Use `dragElastic={0}` to prevent the terminal from being pulled even slightly out of bounds.

### 3. Evidence Visibility in 3D
- **Approach**: Filtering the `clues` array in `InsideRoomArena.tsx` before it reaches the 3D scene graph.
- **Optimization**: Prefer conditional rendering over the `visible` prop to reduce draw calls and scene complexity.
- **State Source**: Use the `is_hidden` field in `TMAEvidence`.
- **Pattern**:
  ```tsx
  {clues.filter(c => !c.is_hidden).map(clue => (
    <EvidenceSprite3D key={clue.id} evidence={clue} ... />
  ))}
  ```

### 4. Double-Voting Prevention
- **Approach**: Local state-based locking in the `PollCard` component within `EvidenceTab.tsx`.
- **Logic**:
  ```tsx
  const [isVoting, setIsVoting] = useState(false);
  const handleVote = async (vote: boolean) => {
    if (isVoting || hasVoted) return;
    setIsVoting(true);
    try {
      await submitVote(...);
      setHasVoted(true);
    } finally {
      setIsVoting(false);
    }
  };
  ```
- **UI**: Disable the buttons and apply `opacity-50` when `isVoting` is true.
